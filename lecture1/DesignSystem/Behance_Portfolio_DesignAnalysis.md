## 1. Typography System

### Font Style 분류
```
세리프 계열      → 고급스럽고 에디토리얼한 느낌  (패션·브랜딩 포트폴리오)
산세리프 계열    → 모던하고 기술적인 느낌        (UX/UI·개발자 포트폴리오)
스크립트/손글씨  → 창의적이고 감성적인 느낌      (그래픽 디자이너)
한글 고딕 볼드   → 국내 디자이너 정체성 강조     (타이포그래피 포트폴리오)
```

### Font Size Scale (관찰값 기준)
```css
/* Display — 히어로 타이틀, Portfolio 단어 자체를 비주얼로 활용 */
--font-display-hero:  80px ~ 200px;   /* 풀스크린 타이포 커버 */
--font-display-title: 60px ~ 80px;    /* 이름 또는 슬로건 메인 타이틀 */

/* Heading — 섹션 제목, 프로젝트명 */
--font-heading-lg: 32px ~ 60px;       /* 프로젝트명, 카테고리명 */
--font-heading-md: 24px ~ 32px;       /* 서브 섹션 제목 */

/* Body — 본문 및 설명 */
--font-body-md: 14px ~ 16px;          /* 프로젝트 설명, 역할 기술 */
--font-body-sm: 12px ~ 14px;          /* 분야 태그, 포지션, 연도 */

/* Caption — 부가 정보 */
--font-caption: 10px ~ 12px;          /* 연락처, URL, 저작권 표기 */
```

| **세리프 스크립트 혼합** | 손글씨 스크립트 + 세리프 혼용으로 개성 표현 | Hyein Jeong, 이지우 |

### Typography 배치 규칙
```
[상단] 분야 카테고리 태그  →  Brand Design · Creative Visual · Graphic Design
[중앙] 메인 타이포 비주얼  →  Portfolio. / 이름 / 슬로건
[하단] 이름 / 연도 / 연락처 →  좌하단 정렬 또는 양끝 정렬
```

## 2. Layout System

### Cover Layout 유형
| 유형 | 사용 비율 | 특징 |
|------|-----------|------|
| **풀블리드 타이포** | ~35% | 텍스트가 화면을 가득 채움, 단색·텍스처 배경 |
| **좌우 분할 (50:50)** | ~20% | 좌측 이미지/비주얼 + 우측 텍스트 정보 |
| **오브젝트 중심** | ~20% | 3D 오브젝트·사진·캐릭터 중앙 배치, 텍스트 주변 배치 |
| **에디토리얼 그리드** | ~15% | 출판물처럼 컬럼 기반 다단 텍스트 배치 |
| **목업/스크린 중심** | ~10% | 실제 작업물 목업 이미지를 커버로 사용 |

### Grid System (관찰값 기준)
```css
/* Behance 검색 결과 카드 그리드 */
--grid-columns-desktop: 4;      /* 데스크톱: 4열 */
--grid-columns-tablet:  2;      /* 태블릿: 2열 */
--grid-columns-mobile:  1;      /* 모바일: 1열 */
--grid-gap:             24px;   /* 카드 간 간격 */

/* 포트폴리오 내부 페이지 */
--inner-grid-2col: 1fr 1fr;     /* 프로젝트 이미지 2단 배치 */
--inner-grid-full: 1fr;         /* 풀페이지 이미지 슬라이드 */
```

### 내부 페이지 레이아웃 패턴
```
TOC 페이지     →  번호 + 카테고리명 리스트 (01 Branding / 02 Editorial...)
풀페이지 슬라이드 →  프로젝트 1건 = 풀스크린 1페이지
2단 그리드     →  좌우 동일 비율 프로젝트 이미지 배치
상단 카테고리바 →  PORTFOLIO | PROMOTION | PRODUCT | BRANDING 고정 표시
섹션 번호 표기  →  01, 02, 03... 또는 1, 2, 3 순서 부여
```

### 섹션 구분 방식
```
방식 A  →  수평선(HR) 또는 넓은 여백으로 분리
방식 B  →  배경색 전환  (흰색 → 검정 → 흰색)
방식 C  →  번호 + 카테고리명  (01 Branding / 02 Editorial)
```

## 4. Component Tokens

### 커버 고정 구성 요소
| 요소 | 위치 | 형태 |
|------|------|------|
| **Name** | 하단 좌측 또는 중앙 | 소형·중형 텍스트 |
| **Year** | 우하단 또는 타이틀 옆 | 2025 / 2026 숫자 |
| **분야 태그** | 상단 | 점(·) 구분 키워드 나열 |
| **연락처/링크** | 하단 | 이메일·인스타그램·링크드인 아이콘 + URL |
| **TOC** | 두 번째 페이지 | 번호 + 프로젝트명 리스트 |

### 커버 텍스트 구조 패턴
```
패턴 A — 타이포 중심
  [Display]  Portfolio.
  [Caption]  이름 / 연도 / 분야

패턴 B — 인사말 중심
  [Heading]  안녕하세요, OOO 입니다.
  [Body]     디자인 분야 소개 한 줄 카피

패턴 C — 키워드 나열
  [Display]  분야1 · 분야2 · 분야3
  [Caption]  이름 / 연락처

패턴 D — 슬로건 중심
  [Display]  견고하게 성장하는 / Think. Make. Reveal.
  [Caption]  이름 / 포지션
```

## 5. Graphic Elements

### 배경 처리 방식
| 방식 | 특징 |
|------|------|
| **단색 배경** | 집중도 높음, 타이포 강조에 적합 |
| **그레인 텍스처** | 빈티지/필름 감성, 레트로 무드 |
| **그라디언트** | 공간감과 깊이감 부여 |
| **포토그래피** | 인물 사진 또는 작업 환경 사진 |
| **일러스트/페인팅** | 강한 개성 표현, 아티스트형 디자이너 |

### 그래픽 모티프
```
원형 (Circle)      →  부드러운 흐름, 무한 반복  (브랜딩 포트폴리오)
격자/그리드선       →  설계·디자인 시스템 중심 포트폴리오
밑줄/라인          →  텍스트 강조, 에디토리얼 구분선
도형 (삼각·사각)    →  구성적 균형감, 기하학적 미감
스티커/뱃지         →  Y2K, 팝 스타일 포트폴리오
```

### 목업(Mockup) 유형
```
브랜딩 포트폴리오   →  명함, 패키지, 굿즈 목업
UX/UI 포트폴리오   →  스마트폰, 태블릿, 노트북 목업
에디토리얼 포트폴리오 →  책, 잡지 목업
```

## 6. Interaction & UX Patterns

```
커버 호버 효과  →  마우스 오버 시 좋아요·저장 아이콘 노출
페이지 전환     →  단일 세로 스크롤로 모든 작업물 순차 감상
외부 링크 배치  →  커버 하단에 이메일·SNS·포트폴리오 URL
채용 연동       →  Behance PRO "채용 가능" 배지 노출
```
