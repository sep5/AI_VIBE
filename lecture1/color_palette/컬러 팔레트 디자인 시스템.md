## 컬러 사용 가이드

### Primary Color — `#C8102E` (레트로 레드)
- **사용 위치**: 포인트 아이콘(별표·꽃), 강조 텍스트, 버튼 호버 상태
- **규칙**: 전체 화면에서 5% 이하로 절제 사용 — 시선을 끄는 포인트 역할에 한정
- **금지**: 배경 전체나 대형 블록에 단독 사용 금지 (시각 피로 유발)

### Secondary Color — `#1A1A1A` (딥 블랙)
- **사용 위치**: 메인 타이포그래피, 다크 카드 배경, CTA 버튼 기본 상태
- **규칙**: 크림 배경(`#F0EBE3`)과 조합 시 고대비 확보 — 가독성 앵커 역할
- **조합**: 다크 카드 배경 위에는 반드시 `--color-text-secondary(#FFFFFF)` 사용

### Accent Color — `#A8C4D8` (파우더 블루)
- **사용 위치**: 카테고리 카드, 태그/칩 배경, 섹션 구분 블록
- **규칙**: `#1A1A1A` 텍스트와 조합해 충분한 명도 대비 확보
- **느낌**: 경쾌하고 읽기 편한 편집 디자인 분위기 형성

### Background Colors
| 변수 | 용도 |
|------|------|
| `--color-bg-primary` | 전체 페이지 기본 배경 — 따뜻한 여백감 |
| `--color-bg-secondary` | 강조 섹션, 풀스크린 다크 배경 |
| `--color-bg-card-blue` | 기술·정보 중심 카드 배경 |
| `--color-bg-card-pink` | 감성·소개 중심 카드 배경 |

### Text Colors
- `--color-text-primary` (`#1A1A1A`): h1~h3, 모든 본문 텍스트
- `--color-text-secondary` (`#FFFFFF`): 다크/블랙 배경 위 텍스트 전용
- `--color-text-muted` (`#6B5B52`): 날짜, 캡션, 부가 설명 — 크림 배경과 자연스럽게 조화

### Border & Outline
- **블루 점선** (`#4A6FA5`): 뱃지, 라벨, 보조 태그 윤곽선
- **핑크 점선** (`#E8909A`): 소프트한 구분 요소, 섹션 콜아웃 박스
- **기본 구분선** (`#DDD5CB`): 카드 사이 여백 구분, HR 요소

### Interactive Colors
- **버튼 기본**: `#1A1A1A` 배경 + `#FFFFFF` 텍스트
- **버튼 호버**: `#C8102E` 배경 전환 (0.2s ease transition 권장)
- **링크**: `#4A6FA5` → 호버 시 `#C8102E` (underline 없이 색상 변화만)

## 반응형 고려사항

### 다크모드 대응
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #1E1A17;       /* 크림 → 다크 브라운으로 전환 */
    --color-bg-secondary: #0D0D0D;
    --color-bg-card-blue: #2A3D4F;     /* 채도 낮춘 다크 블루 */
    --color-bg-card-pink: #4A2530;     /* 채도 낮춘 다크 핑크 */
    --color-text-primary: #F0EBE3;     /* 크림 텍스트 — 배경과 반전 */
    --color-text-muted: #A89A90;
    --color-border-default: #3A3028;
    /* Primary(레드), Accent(블루), Secondary(블랙)는 유지 */
  }
}
```

### 모바일 (xs ~ sm: 0px ~ 600px)
- 콜라주 카드 레이아웃 → 세로 단일 스택으로 전환
- 카드 `border-radius: 12px`, `padding: 16px` 으로 터치 친화적 크기 확보
- 별표 포인트 아이콘은 모바일에서 `font-size: 1rem`으로 축소
- 점선 타원 뱃지는 모바일에서 `display: none` 처리 권장 (레이아웃 혼잡 방지)

### 태블릿 (md: 900px)
- 카드 2열 그리드 배치 (`Grid size={{ xs: 12, md: 6 }}`)
- 타이포그래피 `fontSize: { xs: '2rem', md: '3rem' }` 전환
- 점선 뱃지 복원 및 `position: absolute` 로 카드 주변 배치

### 데스크톱 (lg: 1200px 이상)
- 콜라주 레이아웃 완전 적용 (카드 겹침, 회전 효과)
- `Container maxWidth="lg"` 기준 중앙 정렬
- 카드 hover 시 `transform: translateY(-4px)` + `box-shadow` 추가 권장

### 접근성 (WCAG AA 기준)
| 조합 | 대비율 | 기준 |
|------|--------|------|
| `#1A1A1A` on `#F0EBE3` | 16.8:1 | AA / AAA 충족 |
| `#FFFFFF` on `#1A1A1A` | 20.6:1 | AA / AAA 충족 |
| `#C8102E` on `#F0EBE3` | 5.1:1 | AA 충족 (대형 텍스트) |
| `#1A1A1A` on `#A8C4D8` | 9.2:1 | AA / AAA 충족 |
| `#C8102E` on `#F5A0AA` | 3.1:1 | AA 미달 — 아이콘 전용 사용 |

> `#C8102E` 레드 on 핑크 배경 조합은 텍스트에 사용 금지 — 장식 아이콘에만 허용
