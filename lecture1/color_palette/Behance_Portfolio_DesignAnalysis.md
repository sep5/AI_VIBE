- **세리프 계열**: 고급스럽고 에디토리얼한 느낌 (패션 포트폴리오, 브랜딩 포트폴리오)
- **산세리프 계열**: 모던하고 기술적인 느낌 (UX/UI, 개발자 포트폴리오)
- **스크립트/손글씨**: 창의적이고 감성적인 느낌 (그래픽 디자이너)
- **한글 고딕 볼드**: 국내 디자이너 정체성 강조 (그래픽 디자인, 타이포그래피 포트폴리오)

### 1-3. 타이포 배치 규칙
- 연도(2025, 2026)를 타이포 요소로 나란히 배치해 현재성 강조
- 이름을 좌하단 또는 중앙에 배치해 브랜딩 확립
- 전문 분야 키워드(Brand Design / Creative Visual / Graphic Design)를 상단 가로로 나열

## 2. 레이아웃 구조

### 2-1. 커버 레이아웃 유형
| 유형 | 비율 | 특징 |
|------|------|------|
| **풀블리드 타이포** | ~35% | 텍스트가 화면을 가득 채움. 배경은 단색 또는 단순 텍스처 |
| **좌우 분할 (50:50)** | ~20% | 좌측 이미지/비주얼 + 우측 텍스트 정보 |
| **오브젝트 중심** | ~20% | 3D 오브젝트·사진·캐릭터를 중앙 배치, 텍스트는 주변 배치 |
| **에디토리얼 그리드** | ~15% | 출판물 레이아웃처럼 컬럼 기반 다단 텍스트 배치 |
| **목업/스크린 중심** | ~10% | 실제 작업물 목업 이미지를 커버로 사용 |

### 2-2. 내부 페이지 레이아웃 패턴
- **목차 페이지 (TOC)**: BRAND EXPERIENCE / IDENTITY / STRATEGY / GRAPHIC / EDIT / ILLUST 등 항목을 리스트로 나열
- **섹션 번호 시스템**: 01, 02, 03... 또는 1, 2, 3으로 프로젝트 순서 표기
- **풀페이지 이미지 슬라이드**: 프로젝트 한 건 = 한 페이지 풀스크린
- **2단 그리드**: 좌우 동일 비율로 프로젝트 이미지 배치
- **상단 정보바**: PORTFOLIO | PROMOTION | PRODUCT | BRANDING 등 카테고리 바 고정

### 2-3. 정보 배치 규칙
```
[상단] 분야 카테고리 태그 (Brand Design · Creative Visual · Graphic Design)
[중앙] 메인 비주얼 (타이포 또는 이미지)
[하단] 이름 / 연락처 / 연도 / SNS 링크
```

## 4. 반복 등장하는 디자인 요소

### 4-1. 고정 구성 요소
| 요소 | 위치 | 형태 |
|------|------|------|
| **이름 (Name)** | 커버 하단 좌측 또는 중앙 | 소형 또는 중형 텍스트 |
| **연도 (Year)** | 커버 우하단 또는 타이틀 옆 | 2025 / 2026 숫자 |
| **분야 태그** | 커버 상단 | Brand Design / Graphic / UX/UI 등 점(·)으로 구분 |
| **연락처/링크** | 커버 하단 | 이메일, 인스타그램, 링크드인 아이콘 + URL |
| **목차 (TOC)** | 두 번째 페이지 | 번호 + 프로젝트명 리스트 |

### 4-2. 커버 텍스트 구조 패턴
```
패턴 A — 타이포 중심:
  [대형] Portfolio.
  [소형] 이름 / 연도 / 분야

패턴 B — 인사말 중심:
  [중형] 안녕하세요, OOO 입니다.
  [소형] 디자인 분야 소개 한 줄 카피

패턴 C — 키워드 나열:
  [대형] 분야1 · 분야2 · 분야3
  [소형] 이름 / 연락처

패턴 D — 슬로건 중심:
  [대형] 견고하게 성장하는 / Think. Make. Reveal.
  [소형] 이름 / 포지션
```

### 4-3. 섹션 구분 방식
- 수평선(Horizontal Rule) 또는 여백으로 섹션 분리
- 번호 + 카테고리명 (01 Branding / 02 Editorial...)
- 배경색 전환으로 섹션 구분 (화이트 → 블랙 → 화이트)

## 5. 이미지/그래픽 요소 활용

### 5-1. 배경 처리 방식
| 방식 | 특징 |
|------|------|
| **단색 배경** | 집중도 높음, 타이포 강조에 적합 |
| **그레인 텍스처** | 빈티지/필름 감성, 레트로 무드 |
| **그라디언트** | 공간감과 깊이감 부여 |
| **포토그래피** | 인물 사진 또는 작업 환경 사진 |
| **일러스트/페인팅** | 강한 개성 표현, 아티스트형 디자이너 |

### 5-2. 그래픽 모티프
- **원형(Circle)**: 부드러운 흐름, 무한 반복 느낌 (브랜딩 포트폴리오에서 자주 등장)
- **격자/그리드선**: 디자인 시스템, 설계 중심 포트폴리오
- **밑줄/라인**: 텍스트 강조, 에디토리얼 구분선
- **도형(삼각형·사각형)**: 구성적 균형감, 기하학적 미감
- **스티커/뱃지**: Y2K, 팝 스타일 포트폴리오에서 활용

### 5-3. 목업(Mockup) 사용
- 브랜딩 포트폴리오: 명함, 패키지, 굿즈 목업
- UX/UI 포트폴리오: 스마트폰, 태블릿, 노트북 목업
- 에디토리얼 포트폴리오: 책, 잡지 목업

## 6. 텍스트 계층 구조 (Typography Hierarchy)

```
Level 1 — Display (히어로 타이틀)
  크기: 80px ~ 200px
  역할: 포트폴리오 제목 또는 이름

Level 2 — Heading (섹션 타이틀)
  크기: 32px ~ 60px
  역할: 프로젝트명, 카테고리명

Level 3 — Subheading (서브타이틀)
  크기: 16px ~ 24px
  역할: 분야 태그, 포지션, 연도

Level 4 — Body (본문)
  크기: 12px ~ 16px
  역할: 프로젝트 설명, 역할 기술

Level 5 — Caption (부가정보)
  크기: 10px ~ 12px
  역할: 연락처, URL, 저작권 표기
```

## 7. 인터랙션 / UX 패턴 (Behance 플랫폼 기준)

- **커버 호버 효과**: 마우스 오버 시 좋아요·저장 아이콘 노출
- **페이지 전환**: 단일 스크롤로 모든 작업물을 세로로 순서 감상
- **외부 링크**: 커버 하단에 이메일·SNS·포트폴리오 사이트 URL 배치
- **채용 연동**: Behance PRO는 "채용 가능" 배지 노출
